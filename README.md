# SageBatch Documentation Website

The official documentation site for **SageBatch** — the MS-DOS Batch 4.0 compatible command shell for SageDOS.

## Tech Stack

- **React 19** + **TypeScript 6** + **Vite 8**
- **Lucide React** icons
- **Glassmorphism** dark design system
- **GitHub Pages** deployment

## Development

```bash
npm install
npm run dev      # Start dev server
npm run build    # Build for production
npm run lint     # Lint with Oxlint
npm run preview  # Preview production build
```

## Deployment

Pushes to `main` automatically deploy to GitHub Pages via `.github/workflows/deploy.yml`.

**Live site:** [https://night-traders-dev.github.io/SageBatch-Docs/](https://night-traders-dev.github.io/SageBatch-Docs/)

## Content

Documents SageBatch's architecture, commands, SageLang implementation, SageVM mode, SageDOS integration, and performance benchmarks.
